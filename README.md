# ultimateRAGBootcamp

[Ultimate RAG Bootcamp Using Langchain,LangGraph & Langsmith](https://udemy.com/course/ultimate-rag-bootcamp-using-langchainlanggraph-langsmith)

## Inintialize uv

```bash
uv init
```

## initialize venv

```bash
uv venv
```

## activate venv

May not be reuired as opening a new terminal automatically activates the venv, but if you need to activate it manually, use the following command:

```bash
uv venv activate
```

## Install dependencies

```bash
uv add -r requirements.txt
```

## installing ipykernel

may be rquired for jupyter notebook in vscode supprt.

```bash
uv pip install ipykernel
```
